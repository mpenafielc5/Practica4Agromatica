Este proyecto permite crear y analizar una base de datos genética usando PostgreSQL y Python. 
Incluye scripts para generar las tablas, importar secuencias desde archivos FASTA, calcular métricas biológicas y visualizar los resultados en gráficos. 
Para usarlo, primero se debe crear la base de datos en PostgreSQL ejecutando los archivos SQL en orden. 
Luego se cargan los datos CSV generados desde los archivos FASTA con el script de Python y pgAdmin. 
Después se ejecutan las funciones y vistas SQL para el análisis biológico. 
Finalmente, el archivo consulta_gc_plot.py permite consultar los resultados, exportarlos a un CSV y generar el gráfico comparativo. 
Con esto se obtiene un flujo completo de análisis genético automatizado y reproducible.
